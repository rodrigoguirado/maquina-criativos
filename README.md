# 🏠 Máquina de Criativos — Seazone

> Geração autônoma de criativos de marketing com IA para empreendimentos SPOT.

## O que gera

- **🖼️ Peças Estáticas** — Formato 4:5 (1080×1350) com foto real + logos + copy + score automático
- **🎙️ Vídeos Narrados** — Roteiros completos + geração de vídeo com Kling (fal.ai)
- **🎬 Vídeos Mônica** — Roteiros com Mônica Medeiros + geração de vídeo com Kling
- **🖼️ Imagens IA** — Geração com Flux via fal.ai
- **✅ Validador automático** — Score ponderado em 5 pilares (Hook, Clareza, Argumento, Comercial, Tom)

## Contexto Seazone Embarcado

Tom de voz consultivo • Sem clichê • Baseado em argumento • Números sempre presentes

## Stack

- **Frontend:** Next.js 14 + React 18 + Tailwind CSS
- **IA:** Claude (via OpenRouter) + fal.ai (Flux + Kling)
- **Deploy:** Vercel

## Como usar

```bash
npm install
cp .env.example .env.local
# Edite .env.local com suas chaves
npm run dev
```
